더 알아보기

스터디 운영이 고민이신가요?

그룹 스터디를 진행하고 싶은데 어떻게 시작해야 할 지 막막하신가요?

초록 스터디에서는 스터디를 진행하는데 필요한 학습 자료를 제공합니다.
그리고 초록 스터디 커뮤니티에서 서로의 경험을 공유하고 도움을 받을 수 있습니다.

성공적인 프로젝트 온보딩에 필요한 다양한 지식과 경험을 습득해보세요.

학습 자료

키워드와 실습 자료

초록 스터디는 학습에 필요한 내용을 전부 알려주지 않습니다. 학습해야 할 내용을 스스로 공부할 수 있는 도움만 드립니다. 웹 애플리케이션 제작에 필요한 개념이나 키워드를 문서를 통해 전달하고 이해를 돕기 위한 실습 환경을 제공합니다.

학습 테스트

언어나 프레임워크의 기능을 익히기 위해 학습 테스트를 제공합니다. 제공되는 학습 테스트를 직접 실행해보고 변형해 보면서 기능에 대한 이해도를 향상시켜보세요.

스프링 학습 테스트자바 학습 테스트

미니 프로젝트 요구사항

학습한 내용을 활용해볼 수 있는 미니 프로젝트의 요구사항을 제공합니다. 실습과 학습 테스트를 통해 학습한 내용을 직접 구현해보면서 익힐 수 있습니다.

스터디 주제

자바 기초

자바 언어를 이제 막 시작하는 입문자 대상

- 동작 검증을 위한 단위 테스트
- 간단한 콘솔 애플리케이션 구현
- 유지보수하기 좋은 클린 코드
- 함수형 프로그래밍 문법 활용

자바 심화(준비중)

자바 언어를 풍부하게 활용하고 싶은 사람 대상

- 제네릭 문법 활용
- JCF 프레임워크 동작 이해

스프링 입문

스프링을 처음 접하는 스프링 입문자 대상

- 웹 요청과 응답 처리를 위한 Spring MVC의 기능
- 데이터베이스 접근을 위한 Spring JDBC
- 스프링의 객체 관리 기능을 위한 Spring Core
- 기본적인 스프링 애플리케이션의 구조인 Layered Architecture

스프링 기초

웹 프로젝트를 위한 배경 지식이 필요한 사람

- Spring 기반의 인증을 구현
- 데이터베이스 접근을 위한 Spring Data JPA
- 스프링의 객체 그리고 외부 설정 관리를 위한 Spring Core
- 간단한 애플리케이션 운동을 위한 배포 스크립트

스프링 심화 - 외부 API 호출(준비중)

스프링 웹 애플리케이션에서 외부 호출에 대한 경험이 필요한 사람

- 외부 API 호출 기능
- 서킷 브레이커 활용
- Fake 테스트

학습 자료 활용 방법

그룹 학습

스터디원들로만 구성된 스터디

스터디원들과 함께 학습 테스트와 미션 수행합니다. 페어로 진행하거나 진행 상황을 공유하면서 학습 할 수 있습니다.

멘토 학습

멘토와 스터디원으로 구성된 스터디

학습 테스트와 미션 수행한 내용을 멘토에게 리뷰 요청을 하고, 멘토의 피드백을 통해 학습을 진행합니다.

초록 스터디 참여 방법

초록 스터디는 그룹 형태로 운영되는 스터디를 지원하고 있습니다.
개인적으로 참여를 희망하는 분들에 대한 지원은 준비중이니 조금만 기다려주세요.

현재 초록스터디는 3기가 진행중입니다.
3기까지는 우아한테크코스 수료생 중 스터디 개설을 희망하는 사람을 대상으로 진행하고 있습니다.
컨텐츠가 안정이되면 외부 그룹 대상으로도 확장 계획이 있습니다.


초록 스터디에 대해 궁금한 점이 있거나 참여를 희망하시면 디스코드에 놀러오세요 :)
