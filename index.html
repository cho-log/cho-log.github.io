초록 스터디는 학습 테스트와 미션을 통해 언어나 프레임워크를 쉽고 효과적으로 학습할 수 있는 환경을 만듭니다. 초록 스터디가 제공하는 다양한 학습 컨텐츠를 활용하여 학습을 시작해 보세요.

더 알아보기

스터디 진행이 고민이신가요?

개인 학습이나 그룹 스터디를 진행하고 싶은데 어떻게 시작해야 할 지 막막하신가요?
초록 스터디에서는 스터디를 진행하는데 필요한 학습 자료를 제공합니다.
웹 백엔드 학습에 필요한 다양한 학습 컨텐츠를 통해 웹 애플리케이션 제작에 필요한 기능과 경험을 습득해보세요.

1. 웹 애플리케이션 제작을 위해 필요한 키워드와 실습

초록 스터디는 학습에 필요한 내용을 전부 알려주지 않습니다. 학습해야 할 내용을 스스로 공부할 수 있는 도움만 드립니다. 웹 애플리케이션 제작에 필요한 개념이나 키워드를 문서를 통해 전달하고 이해를 돕기 위한 실습 환경을 제공합니다.

2. 기능을 연습할 수 있는 학습 테스트

언어나 프레임워크의 기능을 익히기 위해 학습 테스트를 제공합니다. 제공되는 학습 테스트를 직접 실행해보고 변형해 보면서 기능에 대한 이해도를 향상시켜보세요.

스프링 학습 테스트 자바 학습 테스트

3. 미니 프로젝트를 만들어 보는 미션 요구사항

학습한 내용을 활용해볼 수 있는 미니 프로젝트의 요구사항을 제공합니다. 실습과 학습 테스트를 통해 학습한 내용을 직접 구현해보면서 익힐 수 있습니다.

학습 주제

스프링

웹 요청/응답 처리부터 배포 등 웹 애플리케이션 개발에 필요한 다양한 주제를 단계별로 제공합니다.

간단한 CRUD API

Spring Boot 기반으로 Spring MVC를 이용하여 웹 요청/응답을 처리하고 JdbcTemplate을 이용하여 데이터베이스에 연동하는 기본적인 웹 애플리케이션을 만듭니다.

Spring MVC와 인증

기본적인 웹 서비스를 개발하기 위해 필요한 인증을 다룹니다. 인증을 위해서 Spring MVC의 추가적인 기능을 학습합니다.

Spring Environment와 배포

기본적인 웹 서비스를 개발하기 위해 필요한 배포를 다룹니다. 간단한 배포에 필요한 스프링 기능(Spring Configuraion, Profile 등)을 다룹니다.

Spring Data JPA

기본적인 Data JPA 적용을 위한 엔티티 매핑과 관계 매핑을 연습합니다.

외부 API 호출

Fake 테스트와 서킷 브레이커 활용한 외부 API를 호출 기능을 연습합니다.

학습 컨텐츠 활용 방법

그룹 학습

학습 테스트와 미션 수행을 페어로 진행하거나 진행 상황을 공유하면서 학습을 한다면 시너지를 만들 수 있습니다.

멘토 학습

멘토가 있다면 미션 수행 후 멘토로부터 코드 리뷰를 받으며 진행할 수 있습니다.

초록 스터디에 참여하는 방법은?

초록 스터디는 그룹 형태로 운영되는 스터디를 지원하고 있습니다.
개인적으로 참여를 희망하는 분들에 대한 지원은 준비중이니 조금만 기다려주세요.

컨텐츠가 안정이되면 외부 그룹 대상으로도 확장을 할 계획입니다.
초록 스터디에 대해 궁금한 점이 있거나 참여를 희망하시면 문의를 남겨주세요 :)
